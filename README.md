# Atividade de Vetores em Portugol

**Módulo:** T0-INTRO-TECH-8 - Prática no Portugol

## Descrição
Algoritmo que lê 10 números inteiros e armazena em um vetor, depois exibe:
- Elementos nos índices ímpares
- Elementos que são números pares
- Soma de todos os elementos
- Média dos elementos

## Como executar
1. Acesse https://portugol.dev
2. Cole o código do arquivo `atividade_vetores.por`
3. Clique em "Executar"
4. Digite os 10 números quando solicitado

## Tecnologias utilizadas
- Portugol Webstudio
- GitHub
